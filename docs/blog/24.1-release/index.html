Document Filters 24.1.0 is now available for download!

Features Demoâ€‹

HD Support for PDF XFAâ€‹

PDF XFA files are often used for interactive forms and dynamic documents in the government, education, and financial industries. Document Filters now allows for rendering these files in high definition, adding to the 100+ file formats supported in HD.

PDF XFA files require the PDF_XFA_ENABLED option, with the default having it disabled.

Comment Extraction from Archive Formatsâ€‹

Document Filters always has a goal of being able to extract every piece of information from files, and weâ€™ve now expanded that. Itâ€™s now possible to extract comments from archive files (ZIP, RAR) with Document Filters.

OLM Date Fieldâ€‹

Document Filters now allows for choosing which OLM date field should be used as the main Date field in OLM files.

The OLM_DATE_FIELD option can be set to a comma separated list of fields that determines the search priority for the OLM Date field. The default is an empty list, which results in the first field that's found being used for the field.

XLS External Workbookâ€‹

Links to external workbooks in XLS files are now processed when extracting text.

When extracting text from an XLS file and the EXCELMODE=text and SHOWHIDDEN=on options are set, the SHOWHIDDEN_EXCEL_REFS option can be enabled to extract links to external workbooks. This option is disabled by default.
